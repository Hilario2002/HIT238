Below are my answers to this quiz.

What is Semantic HTML?

In "Semantic HTML", markup gives emphasis to the meaning (hence the term "semantic") of the content.

How do you import a stylesheet into an HTML document?

The stylesheet is referenced using the <link> element inside the <head> element.
For example: <link rel="stylesheet" href="assets/css/main.css" />

How do you import javascript into an HTML document?

The <script> tag is inserted into the <head> element.
For example: <script type = "text/javascript" src = "myscript.js" ></script>

What meta tag can you use to fix the mobile screen width?

The viewport meta tag gives the browser instructions for controlling the height, width and scale of the web page.
For example: <meta name="viewport" content="width=device-width, initial-scale=1.0">

How do you apply CSS only to certain screen sizes?

@media queries will include a block of CSS only if the condition(s) is true.
For exmaple:


@media only screen and (max-width: 600px) {
	body {
		background-color: rgba(255, 0, 0, 0.3); /* transparent red */
	}
	}

What does "mobile-first" mean when writing CSS?

It is an approach that gives precedence or importance to designing the UX (User Experience), CSS in this instance, for mobile above all else.

How does a browser decide the order to apply CSS?

The priority in which CSS is applied in the following order:

  1. Inline
  2. Internal or embedded
  3. External

Name three javascript best practices.

Some good advice to follow when coding are:

  • Sensibly short variable and function names that are easy to understand
  • Comment, comment, comment but only in moderation
  • Loop optimisation (Freecodecamp, 2019)

How do you run a javascript function when a button is pressed?

The onclick() event is triggered when the user left clicks on a button.
Here is an example from 


<html>
	<head>
		<script>
			<!--
				function sayHello() {
				alert("Demo Alert")
				}
			//-->
		</script>
	</head>
	<body>
		<p>Click the following button and see result</p>
		<form>
			<input type="button" onclick="sayHello()" value="Click" />
		</form>
	</body>
	</html>

What is the difference between == and === in javascript?

== compares two variables while ignoring the datatype of the variable.
=== does the same but checks the datatypes before comparing.
